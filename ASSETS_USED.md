## How to Use
Simply open `landing-page.html` in any web browser. All images are referenced relative to the same directory, so ensure all image files remain in the same folder.

## Styling Notes
- All images are responsive and scale appropriately
- App store buttons maintain their original styling
- Smart service cards use CSS object-fit for proper image display
- Footer logos are centered and sized appropriately
